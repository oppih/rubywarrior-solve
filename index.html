Hello Rubywarrior

Feb 6th, 2012 | Comments

I’ve just wrote an About page. And now I want to write the first post here “Hello Rubywarrior” to show that I have started to work on this :)

Actually, I’m working on Level 8 (beginner) currently and just want to find a way to keep my process so I setup this site. And I also want to find someone who is interested in ruby learning and want to improve skills in this way.

I’ll manage to write this blog in English because most contents of this site would just be simple and easy to underseand :)

OK, good luck with my way on “climbing the rubywarroir tower”.

BTW, I came to find, when I’m using ZSH, the common usage of octopress’s command line has some difference:

rake new_post[title]

This should be:

rake new_post\["Hello Rubywarrior"\]

Or I would only get some error message like:

zsh: no matches found: new_post[Hello Rubywarrior]

Not a big problem, just in case you might encounter the same as I did.
